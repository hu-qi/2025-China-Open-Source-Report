# 开源具身智能篇

## 序言：以开放生态推动具身智能走向真实世界

过去几年，人工智能的发展速度令人振奋，也令人深思。从大语言模型到多模态模型，再到正在快速兴起的具身智能，AI 正在从“理解世界”走向“进入世界”。如果说大语言模型让机器具备了更强的语言理解、知识生成与推理能力，那么具身智能则进一步提出了一个更具挑战性的问题：AI 如何在真实物理环境中感知、决策、行动，并与人类社会和产业系统发生真实连接？

2025 年至 2026 年，具身智能已经成为全球人工智能领域最具活力的发展方向之一。从视觉语言动作模型（VLA）到世界模型（World Model），学术界和产业界都在探索同一个方向：如何让人工智能不再停留于屏幕、文本和虚拟空间，而是真正走向机器人、走向工厂、走向家庭、走向城市运行的复杂现场。

在开源社看来，具身智能的发展，天然需要开放生态的支撑。因为具身智能并不是单一模型、单一算法、单一硬件能够独立完成的技术突破，它涉及机器人本体、传感器、控制系统、模型训练、仿真环境、真实场景数据、任务评测、行业应用等多个环节。任何一个封闭体系，都很难独自覆盖如此复杂的创新链条。只有通过开放模型、开放数据、开放工具、开放评测和开放社区，才能让更多开发者、研究者、企业和用户共同参与进来，形成持续演进的创新网络。

这也是开源社长期关注具身智能的重要原因。开源不仅是一种软件开发方式，更是一种面向复杂技术生态的协作机制。大语言模型的发展已经证明，开放协作能够显著降低创新门槛，加快技术迭代，扩大开发者参与，并促进成果的可验证、可复现和可继承。对于仍处在早期快速演进阶段的具身智能而言，这种开放机制同样重要，甚至更加关键。

正是在这样的背景下，我们关注到了 RoboChallenge。

RoboChallenge 致力于构建全球开放的大规模真机评测平台，通过真实机器人环境、大规模标准化任务和开放评测体系，为全球开发者、研究机构和企业提供公平、透明、可复现的能力验证平台。它所关注的，不只是“谁的模型更强”，更是如何建立一个能够被全球社区共同使用、共同检验、共同改进的具身智能评测基础设施。

在平台建设上，RoboChallenge 坚持开放协作的理念。其评测框架和工具链持续在 GitHub 上开源，评测数据集和相关资源通过 Hugging Face 面向全球社区开放下载和使用。这使得更多开发者不仅可以观看真机评测结果，也能够真正参与到具身智能模型验证和技术复现的过程中。

更重要的是，RoboChallenge 不仅关注真机评测本身，也关注评测之后的经验沉淀和生态建设。平台在榜单体系中持续鼓励参赛团队公开模型权重、训练方法和技术路线，推动优秀成果能够被验证、被复现、被学习、被传播。对于一个新兴技术领域而言，这种机制非常重要。因为真正有价值的技术竞赛，不应只是一次性的排名活动，而应成为推动整个领域共同进步的开放基础设施。

开源社与 RoboChallenge 的连接，也正是在这样的共同理念中逐步形成的。早在 2025 年 COSCon 中国开源年会上，我们便关注到具身智能领域所展现出的巨大潜力，并首次设立具身智能专题论坛。RoboChallenge 也正是在那次论坛上首次面向开源社区公开亮相，引发了来自开发者、研究机构和产业界的广泛关注。随后，RoboChallenge 登陆巴黎 GOSIM Open Source Robotics Forum，与全球开源机器人领域的专家、研究机构和企业展开深入交流。围绕开放评测、开放数据和开放生态的理念，RoboChallenge 获得了国际社区的积极响应，也进一步推动了全球具身智能开放评测场的形成。

进入 2026 年，RoboChallenge 又相继成功举办了 CVPR 与 ICRA 两项国际顶级学术会议官方挑战赛。来自全球高校、科研机构、开源社区以及产业界的团队积极参与，共同探索具身智能模型在真实场景中的能力边界。这不仅提升了 RoboChallenge 在国际学术界和产业界的影响力，也让我们更加清晰地看到：开放评测平台、开源基础模型和开放社区协作，正在成为推动具身智能技术创新和产业发展的关键力量。

因此，在本年度《中国开源年度报告》的筹备过程中，开源社决定邀请 RoboChallenge 组委会共同参与，并将《RoboChallenge 年度报告》作为重要内容收录其中。透过 RoboChallenge 这样真机评测基础设施，我们看到了开源力量打破数字与物理边界的可能。因此，我们希望借由本次专题，全景式地呈现具身智能开放生态的当下版图。我们更期待以此启发业界：具身智能不仅是 AI 技术的又一次延伸，它更是开源力量从代码仓库走向具身智能和真实物理世界的关键入口。

值得一提的是，RoboChallenge 组委会并非由单一机构主导，而是以开放共治为核心理念，鼓励不同背景的合作伙伴共同参与决策与建设。来自机器人本体厂商、具身基础模型团队、科研院所、高校、开源社区以及产业用户的代表，围绕评测标准制定、数据开放规范、开源生态建设和行业应用推广等议题持续协作，努力确保平台保持中立、公平、开放的发展方向。这与开源社长期倡导的“共识、共治、共创”精神高度一致。

开源正在重塑具身智能的发展方式。从开放模型到开放数据，从开放评测到开放社区，越来越多的参与者正在共同构建一个更加开放、透明、国际化的创新生态。本报告收录《RoboChallenge 年度报告》，正是希望通过对平台建设、赛事组织、技术创新与国际合作成果的系统梳理，记录这一新兴领域在开放协作精神推动下所取得的重要进展。

我们相信，未来的具身智能不会由单一机构定义，也不会由少数封闭平台独自塑造。它将在全球开发者、研究者、企业、开源社区和产业用户的共同参与下持续演进。开源社也将继续关注和支持具身智能开放生态的发展，与 RoboChallenge 以及更多全球伙伴携手，推动开放源代码、开放数据、开放评测和开放合作成为具身智能创新的重要基础。

让智能走向真实世界，也让开放协作走向更广阔的未来。

陈阳 开源社联合创始人

章文嵩，LVS创始人，开源社顾问委员会

如果你还在为具身智能模型缺乏一个公平、标准化的“考场”而苦恼，那么 RoboChallenge​ 绝对值得你关注。作为由原力灵机Dexmal与Hugging Face联合推出的全球首个具身智能大规模真机评测平台，它用真实的物理环境和严谨的评测体系，为具身基础模型提供了一块难得的“试金石”。

RoboChallenge​ 平台想获得更大的影响力，需要更多的机器人厂商捐赠真机给RoboChallenge平台，更多的外部参与来贡献测试集，要认可他们的贡献并让他们都有所收获，机器人厂商可以看到更多的模型在真机上运行，外部贡献者能获得成就感。就会有更多的具身智能模型开发者到RoboChallenge平台来测试，测试本身有成本，可以考虑有一点点收费让大家慎重测试，但并不以盈利为目的。很欣喜地看到报告中的数据，RoboChallenge已积累超4万次真机测试，社区覆盖中美新日阿联酋等多国开发者，开源数据集下载量达17K次。更难能可贵的是，它坚持公开评测录像与机器日志，让每一份成功与失败都有迹可循——这不仅是一份“成绩单”，更是一本珍贵的“错题集”。

无论你是研究者、开发者还是爱好者，RoboChallenge都为你打开了一扇通往真实物理世界的大门。欢迎加入这场全球共建的评测生态，一起推动具身智能走进人们的生活！

蒋涛， CSDN 创始人，开源社顾问委员会，GOSIM创始人

具身智能的发展，需要的不仅是更强大的模型，更需要开放、公正、可复现的评测体系。RoboChallenge 正在做的事情，就是为行业构建这样一个基础设施平台。
过去一年，RoboChallenge 在真实机器人评测以及国际赛事合作方面取得了积极进展，为开发者、研究机构和企业提供了统一的技术验证平台。我认为，这种基于真机、面向真实场景的评测模式，对于推动具身智能产业发展具有重要意义。
未来，建议 RoboChallenge 优先拓展法国、德国以及日韩等国际市场，同时吸引 OpenCV、Linux 基金会等国际开源社区专家参与治理与指导，进一步提升全球影响力。GOSIM作为RoboChallenge的组委会成员，我们也会极力推进RoboChallenge的国际合作和开发者生态。 
在商业模式方面，可以借鉴开源社区成熟经验，引入 Token 机制，对特定测试和高并发服务进行适度收费，在保障开放性的同时建立可持续发展的运营模式。
期待 RoboChallenge 持续成长，成为全球具身智能领域具有影响力的开放评测平台。
